# Portfolio Asset Checklist

Use this folder to prepare final portfolio images before uploading them through Portfolio Admin.
Files placed here stay local and are not included in Git commits.

## Export Standard

- [ ] Convert final images to WEBP or JPG in sRGB
- [ ] Keep important subjects inside the center 70% safe area
- [ ] Use 1920 x 1080 px for landscape covers
- [ ] Use 1350 x 1800 px for portrait artwork
- [ ] Use 1080 x 1920 px for Story or TikTok artwork
- [ ] Keep each web image around 2-4 MB or smaller
- [ ] Check spelling, dates, logos, and contact information
- [ ] Put the strongest image first when uploading because it becomes the cover

## Upload Order

1. Open Portfolio Admin with a GitHub token.
2. Select the matching project.
3. Upload images in checklist order.
4. Wait for the GitHub commit confirmation.
5. Open both themes and check the cover crop.
6. Open the project deep link and check the full collage.
